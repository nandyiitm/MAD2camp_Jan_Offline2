# MAD2 project demo app

## To run backend
1. cd backend
2. python -m venv venv // to create virtual envirtonment
3. win: venv/Scripts/activate; linux: source venv/bin/activate // activate virtual environment
4. // only to update/add requirements // pip freeze > requirements.txt
5. pip install -r requrements.txt // install packanges
6. python app.py

## To run frontend
1. cd frontend
2. npm install
3. npm run dev
